# Digitalizaci-n-Escuela-de-Patinaje-Roller-Speed-
Plataforma web monolítica
Nombre: Juan Jose Maya Morales - Juan Pablo Montañez
Descripción: Plataforma Web "Roller Speed"
Indicaciones: 
1.	Base de Datos:
○	Instalar PostgreSQL.
○	Crear una base de datos vacía llamada rollerspeed_db.
2.	Configuración:
○	Editar src/main/resources/application.properties.
○	Actualizar spring.datasource.password con la credencial local.
3.	Ejecución:
○	Abrir terminal en la raíz del proyecto.
○	Ejecutar: .\mvnw.cmd spring-boot:run (Windows) o ./mvnw spring-boot:run (Linux/Mac).
